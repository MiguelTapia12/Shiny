Este proyecto contiene una aplicación Shiny para análisis y visualización de datos.

## Características principales
- Inicialización estable del entorno.
- Módulos refactorizados: selección, trazabilidad, dashboard, cruzamientos, estado de variedad.
- Endurecimiento de consultas SQL.
- Pruebas de humo incluidas en `tests/`.

## Instalación
1. Clonar el repositorio:
   git clone https://github.com/MiguelTapia12/Shiny.git
2. Abrir RStudio en la carpeta del proyecto.
3. Ejecutar:
   shiny::runApp()

## Estructura del proyecto
- R/ ? Código modular de la aplicación.
- global.R ? Configuración global y carga de dependencias.
- tests/ ? Scripts de pruebas de humo y validación.
